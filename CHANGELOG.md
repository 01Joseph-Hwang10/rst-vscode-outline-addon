# Change Log

All notable changes to the "rst-vscode-outline-addon" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.1.1] - 2026-03-17

### Changed

- Expanded README with more detailed uninstallation guidance.

## [1.1.0] - 2026-03-17

### Added

- Activation now leaves a marker in the target `tht13.rst-vscode` extension after patching.

### Changed

- Internal release workflow support for the release agent.

## [1.0.0] - 2026-03-17

### Added

- Initial stable release of `rst-vscode-outline-addon`.
- Automatic patching of the locally installed `tht13.rst-vscode` extension to improve `.rst` outline behavior.
- Bundled reStructuredText syntax and configuration assets derived from `trond-snekvik.simple-rst` to support the patch.
